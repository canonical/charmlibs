# charmlibs.interfaces.http_endpoint

The `http_endpoint` interface library.

To install, add `charmlibs-interfaces-http-endpoint` to your Python dependencies. Then in your Python code, import as:

```py
from charmlibs.interfaces import http_endpoint
```

See the [reference documentation](https://documentation.ubuntu.com/charmlibs/reference/charmlibs/interfaces/http_endpoint) for more.
