# charmlibs.interfaces.temporal_worker_consumer

The `temporal-worker-consumer` interface library.

To install, add `charmlibs-interfaces-temporal-worker-consumer` to your Python dependencies. Then in your Python code, import as:

```py
from charmlibs.interfaces import temporal_worker_consumer
```

See the [reference documentation](https://documentation.ubuntu.com/charmlibs/reference/charmlibs/interfaces/temporal_worker_consumer) for more.
