# charmlibs.interfaces.istio_ingress_config

The `istio-ingress-config` interface library.

To install, add `charmlibs-interfaces-istio-ingress-config` to your Python dependencies. Then in your Python code, import as:

```py
from charmlibs.interfaces import istio_ingress_config
```

See the [reference documentation](https://documentation.ubuntu.com/charmlibs/reference/charmlibs/interfaces/istio_ingress_config) for more.
